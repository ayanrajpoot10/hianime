A RESTful API for scraping anime content from hianime.to

Available Endpoints

GET /api/home
Get homepage content including spotlight, trending, and latest anime
GET /api/search?keyword={query}&page={page}
Search for anime by keyword
GET /api/suggestion?keyword={query}
Get search suggestions for a keyword
GET /api/anime/{id}
Get detailed information about a specific anime
GET /api/episodes/{id}
Get episode list for a specific anime
GET /api/animes/{category}?page={page}
Get anime list by category (most-popular, top-airing, etc.)
GET /api/genre/{genre}?page={page}
Get anime list by genre
GET /api/servers?id={episodeId}
Get available servers for an episode
GET /api/stream?id={episodeId}&type={sub|dub}&server={serverName}
Get streaming links for an episode
GET /api/qtip/{id}
Get quick tooltip information for a specific anime
GET /api/schedule?date={YYYY-MM-DD}&tzOffset={offset}
Get estimated schedule for a specific date with timezone offset
GET /api/next-episode/{id}
Get next episode schedule for a specific anime
GET /api/azlist/{sortOption}?page={page}
Get anime list sorted alphabetically or by other criteria
GET /api/producer/{producer-name}?page={page}
Get anime list by producer/studio name
GET /health
Health check endpoint
GET /api/health
API health check endpoint

HiAnime API - Built with Go
